Electron Build Instructions:
    This will give you simple documented instructions on how to compile my applications for all platforms: (Linux, Windows and Mac) as well as architectures like (X64,Arm64 and Arm7l) unless it is using Electron Castlab for video DRM. There will have different instructions below.
    git clone https://gitlab.com/projectname

    cd projectname

    npm i

    ./node_modules/.bin/electron-packager . --all

    Electron Castlab Build Instructions:
    For my software that requires DRM Widevine please follow these instructions. Note: this does not support Arm7l or Arm64 Linux as it does not have a native Linux Arm version yet.
    git clone https://gitlab.com/projectname

    cd projectname

    npm i

    ./node_modules/.bin/electron-builder --linux --win --x64
