    Youtube Music Quick Media Controls
    Hey everyone, 2 days ago I did a small update for Youtube Music that adds convenient quick media control options to the desktop shortcut on Linux. If you are interested in seeing the git changes than click here
    Note: if you aren't using the AUR package you will need to install playerctl on your distro for the controls to work. Now even if you are multi tasking and need to pause, play, skip song or even go back to the previous song you can easily do so by right clicking the icon on your application menu while it's open and you won't have to switch back to Youtube music to do all this. Sound pretty convenient hey?

    If there is any other applications you think the media controls would be great for than please leave a issue request on the apps git repo.
